# Smart Family Finance System

A secure web application to store and manage family financial documents.

## Features
- User authentication with OTP
- Role-based access (Admin & Member)
- Secure document upload & download
- Centralized financial document storage

## Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Database: SQLite

## How to Run
1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
